# 💰 Financial Dashboard

Um painel de controle financeiro interativo desenvolvido com **HTML, CSS e JavaScript**, integrado com **Google Sheets** como banco de dados real.

## 🚀 Funcionalidades

- Adicionar receitas e despesas em tempo real
- Cálculo automático de saldo total
- Visualização de transações em lista
- Dashboard gráfico com receitas x despesas
- Persistência dos dados usando Google Sheets
